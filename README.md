# MEVN Example

example code for simple mongodb, express.js, vue.js, and node.js stacks

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## API setup
```
cd api/
npm install
npm run dev
```
